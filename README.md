# 🔐 Pwn College - Cryptography

Welcome to my solutions and learning notes for the **Pwn College Cryptography** module.

## 🎯 Goals

- Learn cryptography from the fundamentals.
- Improve Python programming skills.
- Document every challenge with clear explanations.
- Build a reusable cryptography toolkit.

## 📂 Repository Structure

```
challenges/
notes/
scripts/
```

## 📖 Learning Workflow

For each challenge:

1. Understand the cryptographic concept.
2. Learn any missing prerequisites.
3. Write the solution in Python.
4. Document the approach and key takeaways.
5. Commit and push to GitHub.
